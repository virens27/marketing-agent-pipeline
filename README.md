# Marketing Agent Pipeline

## Overview

Marketing Agent Pipeline is a multi-agent AI system that automates the complete B2B outreach workflow from account research to campaign execution.

The system uses multiple AI agents working together to:

1. Research target accounts
2. Identify business challenges
3. Generate personalized outreach content
4. Review and score generated drafts
5. Simulate campaign execution and tracking

All agent outputs are stored in Google Sheets, providing complete visibility into the workflow.

---

## Features

### Research Agent V2

* Analyzes target accounts
* Identifies industry insights
* Generates research summaries
* Stores results in Research Cache

### Analysis Agent V2

* Reviews research findings
* Identifies business challenges
* Determines impact and priority
* Stores results in Challenge Map

### Content Agent V2

* Generates personalized outreach drafts
* Creates subject lines
* Builds value propositions
* Produces call-to-action recommendations
* Stores results in Draft Content

### Review Agent V2

* Evaluates generated drafts
* Provides feedback
* Assigns quality scores
* Approves or requests revisions
* Stores results in Review Queue

### Execution Agent V2

* Simulates campaign launch
* Generates campaign metrics
* Tracks campaign progress
* Stores results in Campaign Tracker

## Setup

### Clone Repository

```bash
git clone <repository-url>
cd marketing-agent-pipeline
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key
SPREADSHEET_ID=your_spreadsheet_id
GOOGLE_SHEETS_CREDENTIALS_FILE=credentials.json
```
